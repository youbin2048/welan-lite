# Welan Lite

`Welan Lite` 是一个基于 `ESP32-C3 + ESPHome` 的本地灯控固件，适合做小型水族灯、氛围灯或定时照明控制。

现在这套版本已经面向日常使用，支持：

- 网页本地控制开关和亮度
- 3 组定时场景
- 正念呼吸灯模式
- 433 无线遥控开关灯
- 网页一键重新对频
- OTA 在线升级

## 适用硬件

- 主控：`ESP32-C3`
- 灯光输出：`GPIO10`
- 433 接收模块：`灵-R1A M5 串口对码版`
- 433 串口输入：模块 `D0 -> ESP32 GPIO3`
- 对频控制输出：`GPIO1`

## 接线说明

### 灯控

- 灯 PWM 输出：`GPIO10`

### 433 接收模块

- `VCC -> 3.3V`
- `GND -> GND`
- `D0 -> GPIO3`

说明：

- 本项目按 `M5 串口模式` 接入 433 模块
- 不使用 `D1/D2/D3` 作为普通高低电平输入
- 这一版不需要给 `GPIO3` 加上拉电阻

### 433 重新对频

如果你的对频脚需要用三极管模拟按键：

- `GPIO1 -> 电阻 -> 三极管基极`
- 三极管发射极 `-> GND`
- 三极管集电极 `-> 模块 KO/K1 对频脚`

## 功能说明

### 1. 主控制

- 网页可直接开关灯
- 亮度范围为 `0-100`
- 亮度已按实际灯具死区做过映射，不是简单线性 PWM

### 2. 定时场景

- 支持 `T1 / T2 / T3` 三组时段
- 每组都可以单独设置：
  - 是否启用
  - 开始时间
  - 结束时间
  - 亮度

### 3. 正念模式

- 网页可进入/退出正念模式
- 当前节奏为：
  - `5 秒渐亮`
  - `最亮停 1 秒`
  - `5 秒渐暗`
  - `最暗不停留，直接进入下一轮`

### 4. 433 无线遥控

- 按一下遥控器，切换灯的亮/灭
- 已针对 `M5` 串口输出做过去重处理
- 长按不会反复误触发

### 5. 无线遥控重新对频

- 网页系统页提供“无线遥控重新对频”按钮
- 点击后，设备会发送对频脉冲
- 然后按一下遥控器按键完成配对

## 使用方式

### 本地网页

设备连上网络后，直接在浏览器打开设备 IP 即可使用。

页面里可以完成：

- 开关灯
- 调亮度
- 设置三组定时
- 开启正念模式
- 433 重新对频
- OTA 升级

### OTA 升级

在网页系统页选择 `.bin` 固件文件即可升级。
